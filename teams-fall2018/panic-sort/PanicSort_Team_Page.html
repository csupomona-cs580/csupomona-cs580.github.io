INSTRUCTIONS

This web application scans a syllabus given by the user and extracts important dates for exams, quizzes, homework, etc and will either sync it with their google calendar or output a calendar file which can be imported into another calendar application.

To use this web app, the user must choose which type of output they want, "Sync with Google" or "Get Offline Calendar". After, the user needs to upload their syllabus file and choose a scanning method. "Searchable PDF" is a normal searching method and "Scanned PDF" uses Google OCR to scan the file for important information. After you submit, the important dates from the syllabus should be added to you Google Calendar if you signed in. If you choose the offline option, the calendar file should be downloaded automatically.

THE TEAM

Panic Sort is team of masters students putting together a web service for students that have trouble staying on top of due dates listed in a syllabus.

The group is named after a hilariously bad and scary algorithm that attempts to perform a quick sort on a group of data within a set amount of moves.
